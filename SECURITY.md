# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Qwen-Agent, please report it responsibly.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please use one of the following methods:

1. **GitHub Security Advisories (preferred):** Use the ["Report a vulnerability"](https://github.com/QwenLM/Qwen-Agent/security/advisories/new) button on the Security tab of this repository.
2. **Email:** If you prefer email, please contact the Qwen team directly.

### What to include

- A description of the vulnerability
- Steps to reproduce (proof of concept)
- The affected component and version
- Any potential impact

### Response timeline

We will acknowledge receipt of your report within **3 business days** and aim to provide a fix or mitigation plan within **30 days**.
